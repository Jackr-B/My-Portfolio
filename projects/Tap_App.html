Patatap App

An interactive app where every key creates an animation
and a different sound

The animations and sounds were integrated from Paper.js and Howler.js libraries and linked to individual key presses

Github Repository: Tap App

  • Click the image to open the application

Development Notes:
  1. The circles are generated at a random location based off the window dimensions allowing it to work on any size screen.
  2. A unique sound and colour was set to each key.
