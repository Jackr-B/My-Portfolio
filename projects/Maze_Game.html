Maze Game

A game built with a focus on Matter.js and recursion

The goal of the game is to get the ball through a randomly generated maze to the finishing square using the WASD keys

Github Repository: Maze Game

  • Click the image to open the application

Development Notes:
  1. The maze algorithm was built from scratch on a canvas with adjustable levels of difficulty.
  2. Conditions were set to prevent the ball from passing through the walls but would result in end of the game upon touching the ending square.
  3. The ball movement was mapped to the WASD keys with smooth motion physics instead of coming to an abrupt stop on key release.
